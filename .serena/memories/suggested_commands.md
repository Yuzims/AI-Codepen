项目常用命令：
- 前端开发：在 FeiShu-Codepen-Frontend 下运行 `npm run start`
- 前端构建：在 FeiShu-Codepen-Frontend 下运行 `npm run build`
- 前端测试：在 FeiShu-Codepen-Frontend 下运行 `npm run test`
- 前端预览：在 FeiShu-Codepen-Frontend 下运行 `npm run preview`
- 后端开发/启动：在 FeiShu-Codepen-Backend 下运行 `npm run dev` 或 `npm run start`
- 后端测试：在 FeiShu-Codepen-Backend 下运行 `npm run test`
- Windows 环境下当前会话使用 Unix shell 语法；Git、ls、pwd、npm 可直接使用。
